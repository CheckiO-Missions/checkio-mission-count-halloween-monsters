На Хелловін ви вирішили знову випустити монстрів за допомогою старого закляття.

Вхідні дані - рядок, який утворюється шляхом складання імен наступних монстрів:

  • frankenstein
  • ghost
  • jack (jack-o'-lantern)
  • mummy
  • skeleton
  • vampire
  • werewolf
  • witch
  • zombie

Знайдіть і порахуйте монстрів у цьому рядку і поверніть їх кількість.

ЗАУВАЖЕННЯ:
  • Відповідь - це minimum можливе значення.
  • Може існувати декілька монстрів одного типу.

Приклади: 

assert count_halloween_monsters('jack') == 1
assert count_halloween_monsters('jaghost') == 2
assert count_halloween_monsters('zombieletwitch') == 3
assert count_halloween_monsters('jackostmmzombiere') == 5
assert count_halloween_monsters('skfrawitchteinolf') == 4

Вхідні дані: Заклинання (рядок A).

Вихідні дані: Кількість монстрів (ціле число).

Передумова:

  • 4 ≤ len(spell) ≤ 50
