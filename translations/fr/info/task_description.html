Pour Halloween, vous décidez de libérer les monstres à l'aide d'un vieux sortilège.

La valeur d'entrée est une chaîne générée par l'empilement du nom du monstre suivant:

  • frankenstein
  • ghost
  • jack (lanterne)
  • mummy
  • skeleton
  • vampire
  • werewolf
  • witch
  • zombie

Trouvez et comptez les monstres dans cette chaîne et renvoyez le nombre.

REMARQUE:
  • La réponse est la valeur minimum possible.
  • Plusieurs monstres du même type peuvent exister.

Exemples: 

assert count_halloween_monsters('jack') == 1
assert count_halloween_monsters('jaghost') == 2
assert count_halloween_monsters('zombieletwitch') == 3
assert count_halloween_monsters('jackostmmzombiere') == 5
assert count_halloween_monsters('skfrawitchteinolf') == 4

Entrée: Le sort (une chaîne de caractères).

Sortie: Le nombre de monstres (Un entier).

Condition préalable:

  • 4 ≤ len(spell) ≤ 50
