Zu Halloween beschließen Sie, die Monster mit einem alten Zauberspruch wieder zu befreien.

Der Eingabewert ist eine Zeichenkette, die durch Stapeln der folgenden Monsternamen erzeugt wird:

  • frankenstein
  • ghost
  • jack (Jack-o'-Lantern)
  • mummy
  • skeleton
  • vampire
  • werewolf
  • witch
  • zombie

Finden und zählen Sie die Monster in dieser Zeichenkette und geben Sie die Anzahl zurück.

HINWEIS:
  • Die Antwort ist der minimum mögliche Wert.
  • Es können mehrere Monster desselben Typs existieren.

Beispiele: 

assert count_halloween_monsters('jack') == 1
assert count_halloween_monsters('jaghost') == 2
assert count_halloween_monsters('zombieletwitch') == 3
assert count_halloween_monsters('jackostmmzombiere') == 5
assert count_halloween_monsters('skfrawitchteinolf') == 4

Eingabe: Der Zauberspruch (Eine Zeichenkette).

Ausgabe: Die Anzahl der Monster (Eine ganze Zahl).

Vorbedingung:

  • 4 ≤ len(Zauberspruch) ≤ 50
